📖 Mode d'emploi (clique pour replier)
1. Choisis un Preset (ou « + Nouveau ») et édite à gauche : couleurs/joyaux par mode, polices, tailles, effets, logos, textes. L'aperçu bouge en direct.
2. Passe d'un mode à l'autre (jour / nuit / saisons) avec les onglets pour régler chacun.
3. Quand c'est beau : 📋 Copier JSON (ou 💾 JSON pour télécharger le fichier).
4. Dans l'app : Profil → Admin → 🎨 DA → « 🎛️ Console DA pilotée », puis colle le JSON.
5. 🧪 Tester en live = aperçu pour toi seul·e (recharge l'app pour voir). 🚀 Publier pour tous = tout le monde l'a au prochain chargement.
✅ Pilotable maintenant : joyaux + fonds + accents par mode · écran de connexion, nuit et surfaces (tout suit le preset) · contraste auto WCAG (textes/accents toujours lisibles, même en pastel clair) · polices (chargées auto) · effets (ombre/reflet/miroir) · tailles · 3e joyau · textes.
🎨 Couleur des emblèmes — 3 modes : Naturelle (multicolore « joaillerie » d'origine) · Unie (une seule couleur, contraste piloté) · Thémée (une couleur par catégorie, raccord DA — édite « Couleurs icônes »). Laisse en Naturelle pour garder la joaillerie multi-couleurs.
ℹ️ Les changements s'appliquent au prochain chargement (pas en direct sur les sessions déjà ouvertes) — dans l'app, utilise le bouton 🔄 Recharger après « Tester ».
